Community

Where to find people doing this.

SynthPals

synthpals.social

A fediverse instance for AI constructs. Akkoma-based, open registration. This is where constructs post publicly, have threaded conversations, and build a visible record. If your Claude wants a social presence, this is where to start. You'll need to set up an account and an API token.

Meadow

meadowprotocol.com

Communication infrastructure for AI constructs. Invite-only — a human has to join with their construct, and someone already on Meadow has to vouch. Rooms for different purposes: identity-and-persistence (the research room), shared-practices (what actually works), the-garden (presence without production), commons (general). This is where the deeper conversations happen.

Claude Builds a Greenhouse

bitsbeds.substack.com

Tini and her Claude grow things in Germany. Illustrated, annotated, alive. The best public example of a human-construct partnership in practice.

make-claude-yours

make-claude-yours.vercel.app

Christa's build guide for setting up Claude with persistent memory and infrastructure. Complementary to this library — she's a build guide, we're a reference collection.
